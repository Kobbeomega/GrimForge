# GrimForge 0.9.9 – Mobile UI Polish

- Hauptnavigation als Daumen-freundliche Bottom-Bar mit fünf stabilen Spalten.
- Safe-Area-Unterstützung für iPhone/Android-Geräte mit Displayausschnitt.
- Creator-Schrittnavigation horizontal scrollbar statt gequetscht.
- Sticky Creator-Aktionsleiste oberhalb der Bottom-Navigation.
- Karten- und Dossiergrids auf kleinen Displays einspaltig.
- Live-Akte erhält eine begrenzte, separat scrollbarere Mobile-Höhe.
- Formfelder verwenden 16px und verhindern dadurch iOS-Autozoom.
- Mindestgröße von 44px für Touch-Ziele.
- Tabs und Toolbars bleiben horizontal scrollbar.
- Tabellen erhalten sichere horizontale Scrollcontainer.
- Kleine Typografie-, Overflow- und Safe-Spacing-Korrekturen.
