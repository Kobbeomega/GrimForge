# GrimForge 0.9.4 — Compendium Paket C

## Neu

- Lokale Favoriten für alle Compendium-Einträge.
- Favoritenfilter pro Kategorie.
- Favoriten bleiben nach einem Neustart im Browser gespeichert.
- Ausrüstung kann direkt aus dem Compendium einem vorhandenen Charakter hinzugefügt werden.
- Bereits vorhandene Compendium-Gegenstände erhöhen automatisch ihre Menge.
- Waffen, Rüstungen, Schilde, Werkzeuge und Verbrauchsgüter werden passend ins Inventar einsortiert.
- Bestätigungsmeldung nach erfolgreicher Übernahme.
- Artwork-Heroes in Ausrüstungs- und Zauberdetails.
- Vorbereitet für individuelle Gegenstands- und Zauberbilder über die Artwork Registry.

## Datenmodell

Das bestehende Archivformat bleibt kompatibel. Compendium-Gegenstände verwenden stabile Inventar-IDs im Format `compendium-<equipmentId>`.
